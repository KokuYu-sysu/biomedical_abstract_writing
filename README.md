# biomedical_abstract_writing
Here are the skills for agent to write the abstract related to biomedical subject.

Here are two skills for different usage:

## 1. biomedical_abstract_writer
Here contains 3 agents: One for writing the original abstract, one for polishing language to decrease AI marks, and one for review

In the workflow:
Agent 1 $\rightarrow$ Agent 2 $\rightarrow$ Agent 3 $\rightarrow$ Agent 1

To use this skill, you should first upload your proposal about the abstract and the main thought, also uploaded the methods and detail results for AI
**Supporting:**
- Figures
- Table
- Markdown

## 2. biomedical_idea_to_abstract
In this skills contains 4 agents: One for collecting the literatures and information, one for writing the original abstract, one for polishing language to decrease AI marks and one for review

In the workflow:
Agent 1 $\rightarrow$ Agent 2 $\rightarrow$ Agent 3 $\rightarrow$ Agent 4 $\rightarrow$ Agent 2

Further should improve the workflow if agent 2 is not satisfied with the information collected by Agent 1 and add a feedback

To use this skill, You should clearly articulate your ideas and topic using natural language, or provide references that interest you.
